#### 4. 척도
1. 범주형 척도(변수) :  예)  성별, 혈액형 - 빈도수 비율 분석이 의미가 있음
2. 서열척도(변수) :  순서가 있는 데이터 예) 학년  - 빈도수 비율 분석이 의미가 있음
3. 등간척도(변수) :  구간으로 나눠진 데이터 예) 만족도  - 절대원점(0)은 없고, 덧셈 뺄셈 연산 가능
4. 비율 척도(연속형변수) :  절대원점(0)이 존재하고 , 산술연산 가능
